# Distributed Systems

A microservices-based distributed application built with Node.js, Docker and Azure.

## Architecture
- Microservices architecture with separate joke, submit and ETL services
- REST APIs with OpenAPI/Swagger documentation
- API Gateway using Kong
- Docker containerisation with Docker Compose
- Terraform infrastructure as code for Azure deployment
- RabbitMQ messaging concepts

## Tech Stack
HTML, CSS, JavaScript, Node.js, Docker, Docker Compose, Kong, Terraform, Azure, REST APIs, SQL

## Services
- joke-service: serves and manages jokes via REST API
- submit-service: handles joke submissions with Swagger docs
- etl-service: Extract Transform Load pipeline
- kong: API gateway configuration
- terraform: infrastructure as code for Azure deployment
